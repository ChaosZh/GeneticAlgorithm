# 遗传算法程序
## generateData.py
运行该文件，可生成动车进站的随机顺序，此数据接入检修模拟程序分析出检修时间。
### 修改generateData.py文件
|信息|变量名|存储位置|
|:--:|:--:|:--:|
|动车的基本运行信息（到哪些车间，需要的时间）|dfbasic|data文件夹|
|a车间数量|a||
|b车间数量|b||
|c车间数量|c||
|d车间数量|d||
|e车间数量|e||
## calculateTime.py
读入顺序表，生成相应的检修时间。进行排序和筛除，传入mutateData进行变异。
## mutateData.py
选择表后给出表序号进行变异。
变异方式有两种：
- 交叉变异（两个表之间车间数据交换）
- 突变（单个表中车间数据突变）
